# 名産マップ

日本の名産品と世界の名産品を地図上で発見できるWebアプリケーションです。

## 🌍 世界版の新機能

- **全世界対応**: アジア、ヨーロッパ、北アメリカ、南アメリカ、アフリカ、オセアニア、中東の名産品を収録
- **大陸別フィルター**: 大陸ごとに名産品を絞り込み
- **国・地域別表示**: 各国の代表的な名産品を表示
- **ズームレベル対応**: 地図のズームレベルに応じて表示する名産品を調整

## 機能

- 🗺️ **インタラクティブな地図**: Leaflet.jsを使用した美しい地図表示
- 🔍 **検索機能**: 名産品名、都道府県、説明文での検索
- 🏷️ **カテゴリフィルター**: 果物、野菜、海産物、肉類などで絞り込み
- 🌸 **季節フィルター**: 春、夏、秋、冬、通年で絞り込み
- 📱 **レスポンシブデザイン**: スマートフォンでも快適に利用可能
- 🎨 **美しいUI**: モダンで直感的なユーザーインターフェース

## 技術スタック

- **フロントエンド**: HTML5, CSS3, JavaScript (ES6+)
- **地図ライブラリ**: Leaflet.js
- **アイコン**: Font Awesome
- **デザイン**: カスタムCSS（グラデーション、アニメーション）

## セットアップ

1. リポジトリをクローンまたはダウンロード
2. 依存関係をインストール:
   ```bash
   npm install
   ```
3. 開発サーバーを起動:
   ```bash
   npm run dev
   ```
   または
   ```bash
   npm start
   ```
4. ブラウザで `http://localhost:3000` にアクセス

## バージョン

- **日本版**: `index.html` - 47都道府県の名産品
- **世界版**: `world-index.html` - 全世界の名産品

## 使用方法

1. **地図で探索**: 地図上のマーカーをクリックして名産品の詳細を確認
2. **検索**: 上部の検索ボックスで名産品や都道府県を検索
3. **フィルター**: カテゴリや季節で絞り込み
4. **一覧表示**: サイドバーで名産品一覧を確認
5. **詳細表示**: 商品をクリックして詳細情報を表示

## データについて

- 47都道府県の名産品データを収録
- 各名産品には以下の情報が含まれます:
  - 名称
  - 都道府県
  - カテゴリ（果物、野菜、海産物、肉類、米・穀物、お菓子、飲み物、その他）
  - 季節（春、夏、秋、冬、通年）
  - 説明文
  - 座標（緯度・経度）

## カスタマイズ

### 新しい名産品を追加

`data.js` ファイルの `foodData` 配列に新しいオブジェクトを追加:

```javascript
{
    name: "商品名",
    prefecture: "都道府県名",
    category: "カテゴリ",
    season: "季節",
    lat: 緯度,
    lng: 経度,
    description: "説明文"
}
```

### カテゴリや色を変更

`script.js` ファイルの `categoryColors` オブジェクトを編集:

```javascript
const categoryColors = {
    '果物': '#ff6b6b',
    '野菜': '#4ecdc4',
    // 他のカテゴリ...
};
```
